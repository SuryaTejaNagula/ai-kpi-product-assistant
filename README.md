# 🚀 AI-Powered KPI Product Assistant

An intelligent product analytics app that transforms raw KPI data into actionable product insights using Python, Streamlit, and OpenAI.

---

## 📌 Problem

Product teams rely on dashboards to monitor KPIs, but dashboards do not explain:
- What changed
- Why it matters
- What actions to take

---

## 💡 Solution

Built an AI-powered assistant that:
- Ingests KPI data
- Visualizes trends
- Detects performance changes
- Generates PM-style insights and recommendations

---

## 🧠 Features

- Upload KPI CSV
- KPI dashboard with trends
- Period-over-period comparison
- AI-generated product insights
- Business recommendations

---

## 📊 Metrics Covered

- Signups
- Activated Users
- DAU / MAU
- Retention (30-day)
- Churn Rate
- Subscription Conversion
- Revenue
- NPS

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- OpenAI API

---

## ⚙️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py```
